Web-App (Browser) verwenden

Installation und Grundkonfiguration der Parsec-App auf einem Computer mit Microsoft Windows.

Mit dem Parsec Web Client können Sie mit einem Google Chrome- oder Chromium-Browser auf Ihren Computer zugreifen, ohne die Parsec-Anwendung herunterladen zu müssen. Melden Sie sich entweder über die Parsec-Homepage an oder greifen Sie direkt auf web.parsec.app zu.
